Поиск сайтов по слову

Веб-приложение для поиска новостей через NewsAPI. Вводите слово - получаете статьи. Понравившиеся можно сохранить – они останутся в браузере даже после перезагрузки.

Как скачать и запустить

Скачайте ZIP: https://github.com/sandres1337/praktika → Code → Download ZIP

Распакуйте архив в любую папку.

Откройте терминал в этой папке.

Выполните: npm install

Создайте файл .env и добавьте в него:

text
NEWS_API_KEY=ваш_ключ_с_newsapi.org
PORT=3000
Запустите: npm start

Откройте браузер и перейдите по адресу: http://localhost:3000

Как пользоваться

Введите слово в поле «Search news...», нажмите «Search» или клавишу Enter.

Нажмите «Save» – новость появится в блоке «Saved».

Нажмите «Delete» – новость удалится из сохранённых.

Сохранённые новости не пропадают после перезагрузки страницы.
